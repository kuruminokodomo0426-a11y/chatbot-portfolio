# ポートフォリオ完成チェックリスト

## 1. 最低限の完成条件

- [ ] `http://localhost:3000` でチャットが動く
- [ ] `相談フォームを開く` ボタンで Google フォームへ遷移する
- [ ] Claude への API 呼び出しが成功する（`POST /api/chat 200`）

## 2. 公開前の確認

- [ ] `.env` が Git に含まれていない
- [ ] API キーを画面共有・スクショで公開していない
- [ ] `SERVICE_CONTEXT` が自分の対象業種に合っている

## 3. 応募用に用意するもの

- [ ] デモURL（Vercel など）
- [ ] GitHub リポジトリURL
- [ ] 1〜2分のデモ動画（Loom / YouTube 限定公開）
- [ ] 案件応募用の短い説明文

## 4. 動画シナリオ（そのまま使用可）

1. トップ表示（対象: 小規模店舗向け）を見せる
2. 次の質問を順に送る
   - `美容室向けに導入する場合の最初のステップを教えてください`
   - `営業時間外の問い合わせを減らすにはどうすればいいですか？`
   - `個別相談したいので、問い合わせ方法を教えてください`
3. `相談フォームを開く` を押して遷移確認
