# ポートフォリオチャット — 運用スキル

このプロジェクトで AI の振る舞いや文言を変えたいときの指針。

## 応答の中身を変える

1. **`SERVICE_CONTEXT`**（`.env`）  
   事業内容・強み・対応範囲・注意事項を書く。ここが Claude の前提になる。

2. **`src/lib/system-prompt.ts`**  
   共通ルール（敬語、禁止事項、フォーム誘導のしかた）を変える場合はここ。

## 見た目を変える

- **`src/app/page.tsx`** … ヘッダー文言
- **`src/app/chat-panel.tsx`** … チャット欄の UI

## 新機能を足すとき

- **会話ログの保存** … データベースと認証が必要。スコープを決めてから。
- **ストリーミング表示** … `/api/chat` をストリーム応答にし、`chat-panel.tsx` の fetch を読み替え。
- **埋め込みウィジェット** … 別バンドル or iframe 化など、配布形態を先に決める。

## 安全

- API キーをフロントに渡さない。
- 入力長は `src/app/api/chat/route.ts` の定数で制限済み。必要なら調整。
